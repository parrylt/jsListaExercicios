Lista de Exercícios – Usando a linguagem JavaScript

Rômulo e Marina

Atividade 1 - Calcule a média de diversas notas digitadas pelo usuário.
Atividade 2 - Faça um programa que entre com cinco números e imprima o quadrado de cada número.
Atividade 3 - Peça ao usuário para digitar 5 números em uma caixa de texto. Verifique qual é o maior número e exiba-o.
Atividade 4 - Peça ao usuário para digitar várias idades. Exiba quantas pessoas são maior de idade (18 anos) e quantas são menores.
Atividade 5 - Crie um programa que entre com os dados de altura e sexo de 5 pessoas. Imprima na tela quantas pessoas são do sexo masculino e quantas pessoas são do sexo feminino. Mostre também qual é a maior altura e se essa altura é de um homem ou uma mulher.
Atividade 6 - Peça ao usuário para digitar vários nomes. Exiba na tela todos os nomes digitados, porém de maneira invertida (do último para o primeiro).
Atividade 7 - Peça para o usuário digitar uma data. Exiba separadamente o dia, o mês e o ano. (Obs.: não necessita de laço de repetição).
Atividade 8 - Faça um programa que calcule a soma dos primeiros 50 números pares.
Atividade 9 - Faça um programa que peça uma nota, entre zero e dez. Mostre uma mensagem caso o valor seja inválido e continue pedindo até que o usuário informe um valor válido.
Atividade 10 - Faça um programa que imprima na tela os números de 1 a 20, um abaixo do outro. Depois modifique o programa para que ele mostre os números um ao lado do outro.
Atividade 11 - Faça um programa que leia 5 números e informe o maior número.
Atividade 12 - Faça um programa que leia 5 números e informe a soma e a média dos números
Atividade 13 - Faça um programa que imprima na tela apenas os números ímpares entre 1 e 50.
Atividade 14 - Faça um programa que receba dois números inteiros e gere os números inteiros que estão no intervalo compreendido por eles.
Atividade 15 - Altere o programa anterior para mostrar no final a soma dos números.
